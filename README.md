
# Запуск проекта

## 1. Настройка переменных окружения
Перейдите в папку `server`

Переименуйте файл `.env.example` в `.env`

При необходимости отредактируйте переменные в файле

## 2. Запуск Docker
Перейдите в корень проекта

Запустите Docker Compose
```sh
docker-compose up
```
